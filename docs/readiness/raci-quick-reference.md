# Readiness RACI Quick Reference

Quick accountability map for recurring readiness activities.

| Activity | R | A | C | I |
| --- | --- | --- | --- | --- |
| Daily blocker sync | `pm-duty` | `pm-duty` | `eng-duty`, `qa-duty` | all duty owners |
| Weekly readiness review | `pm-duty` | `eng-duty` | `sec-duty`, `ops-duty`, `data-duty`, `gtm-duty` | leadership |
| Risk re-scoring | `pm-duty` | `pm-duty` | `sec-duty`, `ops-duty`, `eng-duty` | `gtm-duty` |
| RC command runbook execution | `eng-duty` | `eng-duty` | `qa-duty`, `ops-duty` | `pm-duty` |
| Critical journey manual checks | `qa-duty` | `qa-duty` | `eng-duty`, `pm-duty` | `sec-duty` |
| Incident tabletop drills | `qa-duty` | `ops-duty` | `eng-duty`, `sec-duty` | `pm-duty` |
| Security/privacy sign-off | `sec-duty` | `sec-duty` | `eng-duty`, `data-duty` | `pm-duty` |
| Go/no-go facilitation | `eng-duty` | `eng-duty` | `pm-duty`, `qa-duty`, `sec-duty`, `ops-duty` | leadership |
| Post-decision comms | `pm-duty` | `pm-duty` | `gtm-duty`, `data-duty`, `ops-duty` | all stakeholders |

## Notes

- R = responsible (executes work)
- A = accountable (final decision owner)
- C = consulted (must be included before decision)
- I = informed (kept up-to-date)
