# Policy Exception Review Log

Track all approved, rejected, and expired policy exceptions.

## Status values

- `open` - approved and active
- `expired` - passed expiry without renewal
- `closed` - no longer needed
- `rejected` - not approved

## Review cadence

- Review weekly during active readiness cycle.
- Escalate any expired open exception immediately.
