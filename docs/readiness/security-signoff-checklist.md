# Security Sign-Off Checklist

Complete this checklist before final go/no-go sign-off.

## Control completeness

- [ ] Security control evidence matrix updated for current cycle.
- [ ] No unresolved critical vulnerabilities without approved exception.
- [ ] High vulnerabilities within SLA or approved with expiry.
- [ ] Auth/session control checks passed.
- [ ] Authorization control checks passed.
- [ ] Secret governance status verified.
- [ ] Consent compliance evidence verified.

## Documentation completeness

- [ ] `security-privacy-checklist.md` current.
- [ ] `vulnerability-triage-playbook.md` actions reflected in records.
- [ ] Exceptions logged in `policy-exception-review-log.md`.
- [ ] Evidence references added to `evidence-log.md`.
