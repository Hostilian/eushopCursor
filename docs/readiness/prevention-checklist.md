# Readiness Prevention Checklist

Run this checklist at the start of each week to reduce gate-time surprises.

## Ownership hygiene

- [ ] Role directory updated for all critical roles.
- [ ] Backup owners assigned for all gate owners.
- [ ] Owner action register contains no unowned critical actions.

## Evidence hygiene

- [ ] Evidence log updated within last 7 days.
- [ ] Weekly review file exists for current week.
- [ ] Scorecard has current week row updates.

## Quality hygiene

- [ ] Command baseline rerun done for current week.
- [ ] Critical journey manual checks planned with dates.
- [ ] Drill scenarios set to `ready` with facilitators.

## Risk hygiene

- [ ] Risk register reviewed and re-scored.
- [ ] Exceptions with expired dates are closed or escalated.
- [ ] Top 3 risks have active mitigations and owners.

## Decision hygiene

- [ ] Decision log entries recorded for scope or control changes.
- [ ] Go/no-go dependencies list is current.
- [ ] Next checkpoint date is explicitly set.
