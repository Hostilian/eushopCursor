# Reliability Review Checklist

Run this checklist during weekly reliability review.

## SLO health

- [ ] Current SLO values updated.
- [ ] Breaches detected and logged.
- [ ] Breach responses documented.

## Incident outcomes

- [ ] P0/P1 incidents reviewed.
- [ ] MTTR trend updated.
- [ ] Recurrence risk assessed.

## Error budget

- [ ] Burn status calculated.
- [ ] Release posture adjusted if threshold crossed.
- [ ] Stabilization tasks assigned if needed.

## Evidence and actions

- [ ] Evidence links added to `evidence-log.md`.
- [ ] Action items added/updated in `owner-action-register.md`.
- [ ] Dashboard reliability section refreshed.
