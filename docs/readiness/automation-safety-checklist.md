# Automation Safety Checklist

Use this checklist before introducing or changing readiness-related automation.

## Change preparation

- [ ] Purpose and scope documented.
- [ ] Owner and backup owner assigned.
- [ ] Failure mode identified.
- [ ] Rollback path documented.

## Validation

- [ ] Dry-run completed in non-production flow.
- [ ] Output format verified for evidence compatibility.
- [ ] False-positive/false-negative risk reviewed.

## Operational readiness

- [ ] Monitoring and alert path defined.
- [ ] Escalation owner assigned.
- [ ] Change logged in `automation-change-log.md`.
