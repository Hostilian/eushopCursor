# Readiness Announcement Checklist

Use this checklist before sending major readiness announcements.

## Preconditions

- [ ] Decision artifact is finalized and evidence-linked.
- [ ] Audience list is confirmed.
- [ ] Messaging owner is assigned.

## Message content checks

- [ ] Status is factual and evidence-backed.
- [ ] Scope and limitations are explicit.
- [ ] Actions requested from recipients are clear.
- [ ] Next update time is included.

## Distribution checks

- [ ] Internal channel posted.
- [ ] Leadership channel posted (if required).
- [ ] External/customer channel posted (if required).

## Follow-up

- [ ] Questions/issues captured.
- [ ] Clarifications logged in evidence or decision artifacts.
- [ ] Next update sent on time.
