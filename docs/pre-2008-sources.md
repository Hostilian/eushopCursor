# Pre-2008 demographic and migration references

These publications ground **historical** claims about large-scale cross-border mobility in Europe. They are **not** used to justify modern market size for specialty groceries (see investor deck assumptions separately).

1. **United Nations**, Department of Economic and Social Affairs, Population Division (2006). *Trends in Total Migrant Stock: The 2005 Revision*. Consolidates international migrant stock estimates for mid-2000s reporting rounds.  
   https://population.un.org/publications/files/migration/trends_2005/Trends2005.pdf  

2. **Eurostat** (2008). *Population in Europe 2007: first results* (Statistics in Focus, catalogue number KS-SF-08-081). First EU-27 demographic aggregates for reference year 2007.  
   https://ec.europa.eu/eurostat/en/web/products-statistics-in-focus/-/ks-sf-08-081  

3. **OECD** (2007). *International Migration Outlook 2007* (SOPEMI). Annual outlook on migration flows and integration policies using data available through the mid-2000s.  
   https://www.oecd-ilibrary.org/society/international-migration-outlook-2007_migr_outlook-2007-en  

Eushop’s public **[/sources](https://github.com/eushop/eushop)** page on the deployed web app mirrors this list for footnotes on About and Manifesto.
