# Pre-2008 demographic and migration references

These publications ground **historical** claims about large-scale cross-border mobility in Europe. They are **not** used to justify modern market size for specialty groceries (see investor deck assumptions separately).

1. **United Nations**, Department of Economic and Social Affairs, Population Division (2006). *International Migration Report 2006*.  
   https://www.un.org/en/development/desa/population/publications/pdf/migration/migration-report2006.pdf  

2. **OECD** (2007). *International Migration Outlook 2007* (SOPEMI).  
   https://www.oecd.org/migration/imo/  

3. **Eurostat** (2008 release; reference period 2007). *Population in Europe 2007: first results* (Statistics in Focus 81/2008, KS-SF-08-081).  
   https://ec.europa.eu/eurostat/en/web/products-statistics-in-focus/-/ks-sf-08-081  

The web app mirrors this list at **`/sources`** (`apps/web/src/app/sources/page.tsx`) for footnotes on About, Manifesto, and related marketing pages.
